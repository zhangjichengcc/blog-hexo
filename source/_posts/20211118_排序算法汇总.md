## 低级排序

### 冒泡排序

每次循环找出最大值，排到队列头部

``` js
function sort(arr) {
  for (let i = 0; i < arr.length; i ++) {
    const item = arr[i];
    for (let j = i; i < arr.length; j ++ ) {
      if (item < arr[j]) {
        arr[i] = arr[j];
        arr[j] = item;
      }
    }
  }
  return arr;
}
```
